# GetMeBack HQ — Командный Пульт

Визуальный дашборд команды разработки. Показывает кто чем занят, подтягивая задачи из Yandex Tracker в реальном времени.

## Что внутри

- Карточки разработчиков с CSS-персонажами за рабочими столами
- Задачи сгруппированы по статусам: В работе / Ревью / Тестируется / На паузе
- Разделение по отделам: Backend, Frontend, QA, Дизайн
- Автообновление каждые 60 секунд
- Кликабельные ссылки на задачи в трекере

## Запуск

```bash
# 1. Установить зависимости
npm install

# 2. Создать .env из шаблона и заполнить токен + org ID
cp .env.example .env

# 3. Запустить
npm start
```

Сервер стартует на `http://localhost:3000`.

## Переменные окружения

| Переменная | Описание |
|---|---|
| `TRACKER_OAUTH_TOKEN` | OAuth-токен для Yandex Tracker API |
| `TRACKER_ORG_ID` | ID организации в Yandex Tracker |
| `PORT` | Порт сервера (по умолчанию 3000) |

## Стек

- **Backend**: Node.js + Express
- **Frontend**: Vanilla HTML/CSS/JS (один файл)
- **API**: Yandex Tracker REST API v2
